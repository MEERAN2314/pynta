## üî¨ Uncertain Quantities (NEW)

```python
# Create with uncertainty
measurement = UncertainQuantity(100, 'meter', uncertainty=0.5)
# Or with relative uncertainty
measurement = UncertainQuantity(100, 'meter', relative_uncertainty=0.01)

# Automatic error propagation
a = UncertainQuantity(10, 'meter', uncertainty=0.1)
b = UncertainQuantity(5, 'meter', uncertainty=0.05)
area = a * b  # Uncertainty propagates automatically

# Access properties
print(measurement.value)                # 100
print(measurement.uncertainty)          # 0.5
print(measurement.relative_uncertainty) # 0.005
```

## ‚úÖ Unit Validation (NEW)

```python
# Quick validation
validate_unit('meter')      # True
validate_unit('invalid')    # False

# Get suggestions
suggest_unit('metr')        # ['meter', 'meters']

# Advanced validation
validator = UnitValidator()

# Validate with error message
is_valid, error = validator.validate('meter')

# Check compatibility
is_compat, msg = validator.check_dimensionality('meter', 'foot')

# Simplify units
simplified = validator.simplify_unit('meter * second / second')

# Parse compound units
components = validator.parse_compound_unit('meter/second^2')
```

## ‚ö° Batch Operations (NEW)

```python
quantities = [
    Quantity(100, 'meter'),
    Quantity(1, 'kilometer'),
    Quantity(50, 'foot')
]

# Quick functions
results = convert_batch(quantities, 'meter')

converter = BatchConverter()
results = converter.convert_all(quantities, 'meter')

mean = converter.mean_quantities(quantities)
min_val = converter.min_quantity(quantities)
max_val = converter.max_quantity(quantities)
normalized = converter.normalize_units(quantities, 'meter')

# Dictionary operations
data = {'a': Quantity(100, 'meter'), 'b': Quantity(1, 'km')}
converted = converter.convert_dict(data, 'meter')
```

## üéØ Common Patterns

### Pattern 1: Experimental Measurement
```python
# Measure with uncertainty
length = UncertainQuantity(1.00, 'meter', uncertainty=0.01)
time = UncertainQuantity(2.01, 'second', uncertainty=0.02)

# Calculate with error propagation
result = length / time ** 2

# Format for report
print(format_quantity(result, style='latex'))
```

### Pattern 2: Data Validation
```python
# Validate user input
if validate_unit(user_input):
    quantity = Quantity(value, user_input)
else:
    suggestions = suggest_unit(user_input)
    print(f"Did you mean: {suggestions}?")
```

### Pattern 3: Batch Processing
```python
# Process multiple measurements
measurements = [Quantity(x, 'meter') for x in data]
normalized = convert_batch(measurements, 'kilometer')
mean_value = sum_batch(normalized) / len(normalized)
```

### Pattern 4: Performance Optimization
```python
# Profile to find bottlenecks
enable_profiling()
with get_profiler().profile('data_processing'):
    # Your code
    pass
print_profiling_stats()
```

## üêõ Error Handling

```python
from unifyt import (
    DimensionalityError,
    UnitNotFoundError,
    ConversionError,
    ValidationError
)

try:
    result = quantity1 + quantity2
except DimensionalityError as e:
    print(f"Incompatible units: {e}")

try:
    q = Quantity(100, 'invalid_unit')
except UnitNotFoundError as e:
    suggestions = suggest_unit('invalid_unit')
    print(f"Unknown unit. Did you mean: {suggestions}?")
```
